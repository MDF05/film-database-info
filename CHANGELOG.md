# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-01-11

### Added
- Initial release of the simple film database.
- Search functionality with rotating placeholders.
- Movie card display with posters and titles.
- Modal view for detailed movie information.
- Pagination supports.
- Professional documentation (README, LICENSE, CONTRIBUTING, CODE_OF_CONDUCT, SECURITY).
